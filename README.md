# Local Deploy

1. Install [anaconda](https://www.anaconda.com/). 
2. Create virtual envrionment
```
conda create -n MLMD python=3.9
```
3. Install dependent libraries
```
pip install -r requirements.txt
```
4. run MLMD platform
```
streamlit run HOME.py
```
